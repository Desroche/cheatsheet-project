CSS Table Reference

CSS Borders

Property Values Description
border-style: dotted Defines a dotted border.
border-style: dashed Defines a dashed border.
border-style: solid Defines a solid border.
border-style: none Defines no border.

CSS Margins

Property Values Description
margin-top: auto The browser calculates the margin.
margin-right: length Specifies a margin in px, pt, cm, etc.
margin-bottom: % Specifies a margin in % of the width of the containing element.
margin-left: inherit Specifies that the margin should be inherited from the parent element.
All the above Values can be applied to each Property.

CSS Padding

Property Values Description
padding-top: length Specifies a padding in px, pt, cm, etc.
padding-right: % Specifies a padding in % of the width of the containing element.
padding-bottom: inherit Specifies that the padding should be inherited from the parent element.
padding-left: Each Value may be applied to each Property. Negative Values are not allowed.
